# Dotfiles Repository

this here's my dotfiles repository, version controlled for my ease of management
and made public so anyone can draw inspiration from it. it is pretty small and
basic for now, but i plan to add some really cool functions and bindings soon!

## what to expect

this repo is fish focused as it is my interactive shell of choice. might try zsh too
sometime in the future and if i take a liking to it, i will commit my zsh configuration
as well. its unlikely that ill ever use bash interactively but ill write my scripts in
bash for maximum compatibility. lets hope that, even though there are a thousand pitfalls
to mind if im going down that way.

- expect a lot of fzf, i love fzf!
- nice key bindings and fish aliases / functions
- great fish abbreviations (git abbrs in particular)

im also not sure how much of my energy im putting into ricing my system, i like the default
look and feel of plasma 6. even if i happen to try sway, hyprland or niri, its not happening
in the near future. right now im more focused in getting games to work on my linux.
